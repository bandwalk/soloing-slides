🤖 100% 무인 랜덤 빌딩 검증

우측 폰 화면 하단의 '새로운 랜덤 세계관 빌딩' 버튼을 누르면, 백엔드가 장르와 키워드를 즉석에서 자동 교차 조합하여 새로운 게임 스펙을 주입합니다.

• 플랫폼 엔진: Soloing Worlds (UGC Core)

• 스탯-대사 감수: Picasso / Scribe

• 실시간 렌더링: HTML5 / DOM CSS
